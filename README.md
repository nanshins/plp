## Introduction
Parcelを使用した軽量のランディングページ作成用テンプレートです。  



## Get Started

### Development
```bash
# package install
$ yarn install

# development
$ yarn dev
```

### Build
```bash
# web only
$ yarn build:web

# report only
$ yarn build:reports

# all package build
$ yarn buld
```
